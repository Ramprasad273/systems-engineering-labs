# Shared Utilities & Schemas

This directory contains shared libraries, data contracts, and DevOps configurations utilized across both Data Engineering and AI Engineering domains.

## Structure

- `schemas/`: Centralized data contracts (JSON schemas, Protobuf definitions, Avro schemas) ensuring consistency across ingestion pipelines and model consumers.
- `devops/`: Reusable Docker multi-stage build templates, CI/CD GitHub Actions workflows, and infrastructure scripts.
